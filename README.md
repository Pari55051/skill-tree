# Skill Tree
a skill tree (literal tree) made using html and tailwind css.

## Overview

### Tech Stack
- HTML
- Tailwind CSS
- JavaScript

### Features
- skills as leaves on the branches
- click on a skill to view more details
- a fake submission form for suggesting skills to add
- a navbar + footer
- responsive layout
